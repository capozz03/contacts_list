# Тестовое задание _"Список контактов"_

## Установка и запуск

Версия  [Node.js](https://nodejs.org/) v16.13.2.

Установить все зависимости и запустить команду:

```sh
cd contact_list_ts
npm i
npm run dev
```

## Технологии

- ReactJS
- Redux Toolkit (для лучшей интеграции с TypeScript)
- universal-cookie
- TypeScript
- SCSS
- Ant Design
- JSON-server
- jsonwebtoken
